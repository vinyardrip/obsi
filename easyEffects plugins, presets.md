# 🎧 Настройка звука в Linux (EasyEffects)

Для работы всех модулей **EasyEffects** в Arch Linux необходимо установить наборы плагинов.

| Эффект в EasyEffects | Пакет в Arch (pacman/yay) | Описание |
| :--- | :--- | :--- |
| **Эквалайзер / Компрессор** | `lsp-plugins-lv2` | Основной движок для EQ и динамики |
| **Лимитер / Максимайзер** | `zam-plugins-lv2` | Защита от перегрузок (клиппинга) |
| **Шумоподавление (Mic)** | `rnnoise` | AI-подавление фонового шума |
| **Multiband Compressor** | `calf` | Раздельная компрессия частот |
| **Exciter / Bass Enhancer** | `calf` | Улучшение четкости и баса |
| **Пресеты сообщества** | `easyeffects-presets-git` | Готовые конфиги из AUR |

## 🛠 Быстрая установка

```bash
sudo pacman -S lsp-plugins-lv2 calf zam-plugins-lv2 mda.lv2 rnnoise libebur128
```

## 💡 Полезные команды

*   `easyeffects -q` — полностью закрыть службу для обновления списка плагинов.
*   `easyeffects --hide` — запустить в фоновом режиме.

## 🎧 Настройка Sennheiser HD 280 Pro (AutoEq)

Чтобы наушники звучали максимально детально (по стандарту Harman), не нужно крутить эквалайзер на глаз. В сообществе **AutoEq** уже есть готовые параметры именно для этой модели.

### Как применить пресет:
1.  **Установи пресеты из AUR** (если еще не сделал):  
    ```bash
    yay -S easyeffects-presets-git
    ```
2.  Открой **EasyEffects** -> вкладка **Presets** (Пресеты).
3.  Там должен появиться огромный список. Ищи в поиске: `Sennheiser HD 280 Pro`.
4.  Выбери пресет от **rtings** или **oratory1990** (они самые точные).

### Что это изменит:
*   **Суб-бас:** Немного поднимется самый низ ($< 60$ Гц), чтобы наушники звучали мощнее, но без «гула».
*   **Середина:** Уберется легкая «коробочность» звука.
*   **Верха:** Сгладятся резкие пики, звук станет мягче, но детальнее.

---

## 🔊 А что с акустикой?

Для колонок готовые пресеты из интернета **не подходят**, так как звук колонок на 70% зависит от твоей комнаты (отражения от стен, мебели).

> **Совет:** Для акустики в EasyEffects лучше всего использовать модуль **Convolver** (Конвольвер). Но для него нужен измерительный микрофон. 
>
> Если микрофона нет, рекомендую просто добавить **«Эквалайзер»** и на слух чуть-чуть поднять края диапазона (V-образная кривая), если звук кажется плоским.
