### Краткая шпаргалка

| Задача                                        | Команда                                |
| --------------------------------------------- | -------------------------------------- |
| Показать **все** пакеты                       | `pacman -Q`                            |
| Проверить **конкретный** пакет                | `pacman -Q <имя_пакета>`               |
| Показать пакеты, установленные **вручную**    | `pacman -Qe`                           |
| Показать только **зависимости**               | `pacman -Qd`                           |
| Показать пакеты из **AUR**                    | `pacman -Qm`                           |
| **Подробная информация** о пакете             | `pacman -Qi <имя_пакета>`              |
| Узнать, **какому пакету принадлежит файл**    | `pacman -Qo <путь_к_файлу>`            |
